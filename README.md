# AirfoilPrep.jl

Julia interface to SNOPT (must obtain a licensed copy of SNOPT separately).

### To Install

1. Clone the repo:
```julia
Pkg.clone(...)
```

## Run tests

```julia
Pkg.test("AirfoilPrep")
```

## To Use

```julia
using AirfoilPrep
```

See examples in tests.
